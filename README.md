# PROJEKT_MASKA
Projekt Maska został stworzony w ramach konkursu "Sztuczna inteligencja w moim życiu". Przedstawiona jest w nim koncepcja wykrywania czy osoba ma założona maseczkę czy nie. Koncepcja może mieć zastosowanie np. w marketach, na lotniskach czy też przy wejściach do galerii, aby zaalarmować o założenie owej. Projekt został stworzony w oparciu o pakiet Tensorflow.
